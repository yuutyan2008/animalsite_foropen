## 実装の工夫

### 非同期処理

内容投稿フォームで出典URLを入力した際、ページ遷移なしにサイト名を自動取得

- フロントエンド（Alpine.js）がURLの入力完了を検知し、Laravelにリクエストを送信
- Laravelが対象URLにアクセスしHTMLの `<title>` タグを取得してJSONで返す
- サイト名を表示

### セキュリティ対策

上記のURL取得機能はサーバーが外部URLにアクセスする構造のため、SSRF（Server-Side Request Forgery）攻撃の対象になります。
悪意あるユーザーにより、サーバーの認証情報が漏洩するリスクの対策として、プライベートIPおよび予約済みIPへのアクセスをブロックする処理を入れています。
