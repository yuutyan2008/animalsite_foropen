# Solvedience（ソルベディエンス）

犬や猫をめぐる社会問題を理解する目的のサイトです。規定のテーマに関するユーザーの投稿機能があります。
 **公開サイト**: [https://animal-portal-u47f.onrender.com](https://animal-portal-u47f.onrender.com)

## 主要機能

**一般ユーザー**: Googleログイン / 内容投稿（事実・意見の分離、出典URL自動取得） / 自分の投稿管理 / 違反報告 / 退会

**管理者**: 投稿承認ワークフロー / テーマ・カテゴリ・項目のCRUD・ドラッグ&ドロップ並び替え / CSV一括インポート・エクスポート / ユーザー管理（BAN含む） / 違反報告対応 / 要出典リスト・メール催促 / 編集履歴・ロールバック

## 実装の工夫

### 1. ドメインモデル：事実と意見を分ける構造

「事実・根拠」と「考察・意見」を文章ごとにタイプ分けし、出典URLを必須化しました。

```
Theme → Category → Item → Content → ContentSentence
                                         ├ type: 'reference'（事実）/ 'opinion'(意見)
                                         └ url（出典URL）
```

### 2. 編集履歴とロールバック

`content_histories` / `item_histories` テーブルに `created` / `updated` / `deleted` されたデータをJSONで保存し、過去のバージョンに復元できる仕組みを実装しました。


### 3. 退会・BAN・再入会防止

通常退会は個人情報消去後にソフトデリート、BAN退会では `banned_google_ids` テーブルにgoogle_idを残して再入会を防止する設計にしました。


### 4. 非同期処理

内容投稿フォームで出典URLを入力した際、ページ遷移なしにサイト名を自動取得

- フロントエンド（Alpine.js）がURLの入力完了を検知し、Laravelにリクエストを送信
- Laravelが対象URLにアクセスしHTMLの `<title>` タグを取得してJSONで返す
- サイト名を表示

### 5. セキュリティ対策

上記のURL取得機能はサーバーが外部URLにアクセスする構造のため、SSRF（Server-Side Request Forgery）攻撃の対象になります。
悪意あるユーザーにより、サーバーの認証情報が漏洩するリスクの対策として、プライベートIPおよび予約済みIPへのアクセスをブロックする処理を入れています。
